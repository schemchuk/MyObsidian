## Основи
- **Docker** — це платформа для створення, розгортання та запуску контейнерів.
- **[[контейнер]]** — ізольоване середовище для виконання додатків, схоже на віртуальну машину, але легше та швидше.

## Ключові поняття
- **Image (образ)** — шаблон для створення контейнерів.
- **Container (контейнер)** — запущений екземпляр образу.
- **Dockerfile** — інструкції для створення образу.
- **Registry** — репозиторій для зберігання образів (наприклад, Docker Hub).

## Основні команди
```bash
docker ps               # Список запущених контейнерів
docker ps -a            # Список усіх контейнерів
docker images           # Список образів
docker run -it ubuntu   # Запуск контейнера з образом Ubuntu
docker stop <id>        # Зупинити контейнер
docker rm <id>          # Видалити контейнер
docker rmi <image>      # Видалити образ
```

## Переваги
- Легка міграція додатків
- Швидке розгортання
- Менше ресурсів, ніж у віртуальних машин
