## 1. Підготовка
- Використовуємо Kali/Parrot OS.
- Переконатися, що [[nmap]] встановлений:
```bash
sudo apt update
sudo apt install nmap -y
```
- Якщо потрібні NSE-скрипти для веб-пентесту:
```bash
ls /usr/share/nmap/scripts/http-vuln*
```

---

## 2. Перевірка доступності хоста (ping)
```bash
ping -c 4 192.168.1.18
```
- `-c 4` → відправити 4 пакети.
- Якщо ICMP заблокований, пропускаємо до TCP-скану.

---

## 3. Базове [[сканування]] портів
```bash
nmap -Pn 192.168.1.18
```
- `-Pn` → ігнорує пінг, перевіряє порти без ICMP.

---

## 4. Сканування конкретних портів
```bash
nmap -p 80,443 192.168.1.18
```
- Перевірка стандартних веб-портів.

---

## 5. Сканування всіх портів
```bash
nmap -p- 192.168.1.18
```
- Шукаємо нетипові порти (RDP, SMB, SSH, FTP тощо).

---

## 6. Визначення сервісів і версій
```bash
nmap -sV -p- 192.168.1.18
```
- `-sV` → визначає версію сервісу на відкритому порті.

---

## 7. Використання NSE-скриптів для вразливостей
### 7.1 [[HTTP]]/HTTPS
```bash
nmap --script http-vuln* -p 80,443 192.168.1.18
```

### 7.2 SMB
```bash
nmap --script smb-vuln* -p 139,445 192.168.1.18
```

### 7.3 Загальні вразливості
```bash
nmap --script vuln 192.168.1.18
```

---

## 8. Сканування операційної системи
```bash
nmap -O 192.168.1.18
```

---

## 9. Перевірка на firewall
```bash
nmap -sA -p 80,443,3389 192.168.1.18
```

---

## 10. Агреговане сканування
```bash
nmap -sS -sV -O --script vuln -p- 192.168.1.18
```

---

## 11. Логування результатів
```bash
nmap -oN scan_results.txt -sS -sV -O --script vuln -p- 192.168.1.18
```

---

## 12. Рекомендації
1. Починати зі стелс-скану (`-sS`).  
2. Використовувати `-Pn`, якщо ICMP блокується.  
3. NSE-скрипти тільки після відкритих портів.  
4. Для веб-пентесту можна запускати Burp Suite після відкриття HTTP/HTTPS портів.  

---

## 14. Підсумок

- Використовуй покроковий план для тестових машин.  
- Перевір всі порти та сервіси.  
- `filtered` → часто фаєрвол/EDR.  
- NSE-скрипти застосовуй тільки після відкритих портів.  
- Зберігай результати в файл для подальшого аналізу.
