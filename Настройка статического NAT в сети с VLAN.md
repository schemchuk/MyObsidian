### розділяю мережі

для цього використовується статичний NAT
він заключається в тому , що кожній  ІР адресі внутрішнього устройства(приладу) буде співставлений один ІР адрес із пулу зовнішніх устройств
**R2**
R2>en
R2#sh run
щоб приховати внутрішню мережу  з комп РС-3 та РС-4
router rip
do sh ip ro
і тут відміняю процес анонсування (ось строка із таблиці маршрутизації : C 192.168.121.0/24 is directly connected, FastEthernet0/0.121)
no network 192.168.121.0
do wr
**внутрішня мережа забралася з таблиці маршрутизації**
тепер потрібно забезпечити доступ устройств до серверів

### закріплення за кожним внутрішнім ІР адресом пристрою визначеної зовнішьої ІЗ адреси, чка і буде передаватися по мережі
ip nat inside source static 192.168.121.10 10.1.11.2
після чого потрібно сказати для NAT який з інтерфейсів є внутрішнім а який зовнішнім
int fa 0/0.121
ip nat inside    Вказую внутрішній інтерфуйс
ex
int eth 0/1/0.11
ip nat outside   Вказую зовнішній інтерфуйс
ex
do wr
з РС-4 пакет не доходить. тому налаштовую далі на **R2** 

**ip nat inside source static 192.168.121.11 10.1.11.3**  вибираю інший адрес мережі 10.1.11.0 (я можу вибирати від 10.1.11.2(було вибрано для РС-3) до 10.1.11.5  вибираю 10.1.11.3 )
ip nat inside source static 192.168.121.11 10.1.11.3
do wr
вказувати зовнішню і внутрішню мережі не треба
