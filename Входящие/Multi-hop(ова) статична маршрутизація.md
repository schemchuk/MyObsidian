змінюю ІР інтерфейсу interface FastEthernet0/1
для цього заходжу в interface FastEthernet0/1
interface fastEthernet 0/1
ip address 192.168.11.1 255.255.255.0
no sh
do wr 
do sh run подивитись інтерфейс
переходжу до налаштування маршрутизатораR 2, увійшовши в режим глобальної конфігурації. Цей иаршрутизатор доданий і немає налаштуваннь
hostname R2
no ip domain lookup
interface fastEthernet 0/1
no shutdown
ip address 192.168.11.2 255.255.255.0
do wr
int fa 0/0  Перемикнувся між інтерфейсами не виходячи 
no sh
ip address 192.168.12.2 255.255.255.0
do wr
переходжу до налаштування маршрутизатора R3, увійшовши в режим глобальної конфігурації. Цей mаршрутизатор доданий і немає налаштуваннь
hostname R3
no ip domain lookup
interface fastEthernet 0/0
no shutdown
ip address 192.168.12.1 255.255.255.0
do wr
int fa 0/1  Перемикнувся між інтерфейсами не виходячи 
no sh
ip address 192.168.13.1 255.255.255.0
do wr
