задача : переналаштувати підключення із статичною маршрутизвцією в динамічну маршрутизацію. для цього внутрішня мережа маршрутизаторів переведеться на безкласову маршрутизацію, а кінцеві клієнти залишаться класовими.
в якості сіті класу **С** 
 буду використовувати
 192.168.150.0/24 в якій буде пять підмереж кожна з підмереж матиме два хости.
 192.168.150.0/24   це **мережа**
 **підмережі**
 192.168.150.0/30
 192.168.150.4/30
 192.168.150.8/30
 192.168.150.12/30

 після коригування плану переходжу до налаштуваннь маршрутизаторів
 перше це скинути налаштування маршрутизаторів
 заходжу на **R1** в привілейований режим(не в реж гл конф)
 en
 **erase startup-config**  очищати цією командою можна налаштування маршрутизаторів і комутаторів
 далі **перрезапустити** пристрій
 reload  вибрати **n**
 Виконую по черзі на всіх маршрутизаторах
 переходжу до налаштування маршрутизаторів по новому 
 в режимі гл конф
 R1
 hostname R1
 no ip domain-lookup
 int fa 0/0
 no sh
 ip address 192.168.100.1 255.255.255.0
 int fa 0/1
 no sh
 ip address 192.168.150.1 255.255.255.252
 do wr
 interface ethernet 0/1/0
 no sh
 ip address 192.168.150.18 255.255.255.252
 do wr
 **R2**
 hostname R2
 no ip domain-lookup
  int fa 0/1
 no sh
 ip address 192.168.150.2 255.255.255.252
 do wr
 ip address 192.168.150.5 255.255.255.252
 do wr
 **R3**
 hostname R3
 no ip domain-lookup
  int fa 0/0
 no sh 
 ip address 192.168.150.6 255.255.255.252
 int fa 0/1
 no sh
 ip address 192.168.150.9 255.255.255.252
 do wr
  **R0**
 hostname R0
 no ip domain-lookup
  int fa 0/1
 no sh 
 ip address 192.168.150.10 255.255.255.252
  int fa 0/0
 no sh  
 ip address 192.168.50.1 255.255.255.0
 do wr
   int eth 0/0/0
 no sh
 ip address 192.168.150.13 255.255.255.252
 do wr
  **R4**

 no ip domain-lookup
 interface ethernet 0/0/0
 no sh

  interface ethernet 0/1/0
 no sh
ip address 192.168.150.17 255.255.255.252
do wr
