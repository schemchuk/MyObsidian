класс этого бота насдледудуется от **TelegramLongPollingBot** :
public class MyFirstTelegramBot extends TelegramLongPollingBot 
чтобы сгенерировать **токен** переходим в телеграмм и находим через поисковую строку **BotFather** 
вводим сообщение и разбуживаем бота. Он выдает менюшку что он делать может
для создания нового бота кликаем **newbot** 
после введения *уникального* имени для нашего бота выдается менюшка с нужными параметрами
