## связь один к одному
  код в классе Cart
  @OneToOne  
@JoinColumn(name = "UserID", referencedColumnName = "**userID**")  
private Users user;

код в классе Users
@OneToOne(mappedBy = "user", cascade = CascadeType.ALL)  
private Cart cart;

1. поле **userID** в классе Cart связывается с полем **userID** первичного ключа класса Users
2. поле user в классе Users связывается с полем card
