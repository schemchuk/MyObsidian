[[HTTP]] протокол базируется на запросеот клиента серверу

**клиент**  тот кто может выполнить [[HTTP]] **[[запрос]]**  // В качестве **клиента** выступает **браузер** 
сервер программа которая может получить запрос и отправить **[[ответ]]** // В качестве **сервера** выступает некий  **веб-сервер** который умеет отвечать на запросы
в проэкте сервером выступает **джарник** который умеет отвечать на запросы со стороны браузера 
в этом случае на моем компютере поднимается **томкат** сервис и мой комп становится сервером
## запрос
первый рядок - информация о методе, адресе и версии [[HTTP]] протокола
{**метод**} {**адрес** } {  **НТТР и версия**}

после этого идет **заголовок** в формате:
**ключ** : **значение** 
если есть **тело** то оно печатается после **заголовков** и отделяется от них пустым рядком
GET /wiki/Main_Page HTTP
Host : en.wikipendia.org

POST /pages HTTP/1.1
Host : exsample.com
User-Agent : firefox/5.0( Linux; Debian 5.0.8)

## ответ
в первом рядке ответа идет информация об версии НТТР , код состояния и опись состояния
{версии НТТР } {код состояния } {опись состояния } 
HTTP/1.1 200 OK
Server : Apache
Content-Type : text/html; .................
Content- Length : 1234
body answer.(тело ответа)
