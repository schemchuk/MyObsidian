**find -name "*.jpg"** але краще вказати шлях для пошуку 
find /usr/include -name "*.c"  
find /dev -type b знайде всі блочні устройства. це пошук по **типу** **type**
 **ключ f** find /dev -type f знайде файли
 **ключ d** find /dev -type d знайде директоріі
 **ключ b** find /dev -type b знайде всі блочні устройства
sudo find /home/ -user opanas -**знайде всі файли користувача(opanas)** в директорії /home

## з знайденими файлами можна виконувати дії за допомогою команди exec

 find root -name ".txt" -exec cp {} /home/woshe/Desktop/dir1 \;
