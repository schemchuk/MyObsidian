### 3. Ключевая информация для входа

Теперь вы можете использовать следующие данные для входа в ваше приложение:

#### Администратор:

- **Имя пользователя:** admin
- **Пароль:** adminpassword
- **Роль:** ADMIN

#### Обычный пользователь:

- **Имя пользователя:** user
- **Пароль:** userpassword
- **Роль:** TRIAL
