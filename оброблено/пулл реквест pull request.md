## Для того чтобы объединить изменения из удалённой ветки в основную, выполните следующие шаги:

1. **Переключитесь на основную ветку:**
    `git checkout main`
    git checkout main
    
2. **Обновите основную ветку, чтобы убедиться, что у вас актуальная версия:**

    `git pull origin main`
    git pull origin main
    
3. **Слейте изменения из удалённой ветки в основную:**
    
    Сначала подтяните изменения из удалённой ветки:

    git fetch origin имя_ветки
    git fetch origin имя_ветки
    
    Затем выполните слияние:

    `git merge origin/имя_ветки`
git merge origin/имя_ветки`
    
    Если в процессе слияния возникнут конфликты, вам нужно будет их разрешить вручную. После разрешения конфликтов выполните:
    
    git add . git commit
    git add .
    git commit
    
4. **Отправьте изменения в удалённый репозиторий:**
    
    `git push origin main`
    git push origin main
