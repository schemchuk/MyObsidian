3 Теги : [[Основні команди Msfconsole]]
## заходимо в фреймворк 
msfconsole
тут можна виконувати різні команди лінуксу, але **ifconfig** не працює. 
Треба користуватися 
**sudo ip a**
sudo ip a
## команди метасплоіт:
show перечислити будь-який тип модулів, які ми хочемо
use  + модуль -  запустити модуль:
use exploit/windows/[[HTTP]]/sap_host_control_cmd_exec  
після того як зайшли в експлоіт треба ввести 
show info - подивитися інформацію по екстлоіту
show options подивитися параметри
для встановлення корисної нагрузки :
set payload   + додати скопійовану корисну нагрузку
## для зміни параметрів використовується команда:
set
## після налаштуваннь вводиться команда для виконання:
exploit
