ping etf.bg.ac.rs

## Пасивні
## інсрумент WhatWeb
whatweb далі вводяться дані сайту
краще приміняти з ключом -v:
whatweb https://youtube.com/shorts/RtA29Dg9EWO?si=Vv-ueY_Bu_9P1ab_
 -v
після [[сканування]] ifconfig можна побачити що при масці netmask 255.255.255.0  може мінятися в домашній сіті лише останній октет ip адреси inet 192.168.0.145 тобто діапазон сіті складає від 192.168.0.0 до  192.168.0.255
проскануємо **свою** сіть з третім рівнем агресії:
 whatweb 192.168.0.0-192.168.0.255 --aggression 3 -v 
щоб не виводився довгий список помилок можна використати :
whatweb 192.168.0.0-192.168.0.255 --aggression 3 -v --no-errors
тут можна знайти пароль для маршрутизатора і методом перебору встановити пароль входу. 

записати отримані результати в файл по шляху:
whatweb 192.168.0.0-192.168.0.255 --aggression 3 -v --no-errors --log-verbose=/home/kali/Desktop/results
