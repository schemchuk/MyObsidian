за подключение  к **базам данных** отвечает **JDBC** driver manager
## **JDBC** driver manager
YDBC - java database Conektivity

###  он управляет
- подключение к базе данных
- выдачу запросов и команд
- обработкой и получением данных из базы

**DriverManager**  это синглтон содержащий информацию о всех зарегестрированых драйверах

**JDBC**  имеет три основных [[интерфейса]] 
-   **CONNECTION** подключение к базе данных
-  **STATEMENT** выдачу запросов и команд
-  **ResultSet** обработкой и получением данных из базы
эти соединения **нужно** закрывать. можно использовать **try with recours**ed
