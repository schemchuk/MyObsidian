#### спершу треба вияснити скільки хостів активно і їх ІР адреси
треба діяти так ніби сканується домашня сіть 
визначаємо скільки хостів активовано:
виходячи з даних ifconfig :  inet 192.168.0.145  netmask 255.255.255.0  broadcast 192.168.0.2
в мене може мінятися лише остання група цифр.
для виявлення хостів в сіті використовуються пакети (шнструмент): 
**ARP** 
sudo arp -a
деколи треба пропінгувати хост першого :
ping 192.168.0.1
щоб зявилися решта хостів.
потім знову запускаєм :
sudo arp -a

##  кращий інструмент netdiscover :
sudo netdiscover
самостійно знайде всі устройства в сіті 
щоб окремо подивитись ІР роутера :
netstat -nr
