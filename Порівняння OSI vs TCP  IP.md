# **OSI**

**O**pen **S**ystems **I**nterconnection

# **TCP**

**T**ransmission **C**ontrol **P**rotocol

# **IP**

**I**nternet **P**rotocol

| Рівень OSI (7)                      | Функції                                                | Приклади протоколів                                                         | Відповідний рівень TCP/IP (4) |
| ----------------------------------- | ------------------------------------------------------ | --------------------------------------------------------------------------- | ----------------------------- |
| **7. Application** (Прикладний)     | Інтерфейс для користувача, сервіси, доступ до ресурсів | [[HTTP]], HTTPS, FTP, SMTP, DNS, SSH, SNMP( Simple Network Management Protocol) | **Application**               |
| **6. Presentation** (Представлення) | Форматування даних, шифрування, стиснення              | TLS/SSL, JPEG, MPEG                                                         | **Application**               |
| **5. Session** (Сеансовий)          | Керування сесіями, діалогами, синхронізація            | RPC, NetBIOS, PPTP                                                          | **Application**               |
| **4. Transport** (Транспортний)     | Доставка даних, контроль цілісності, сегментація       | TCP, UDP                                                                    | **Transport**                 |
| **3. Network** (Мережевий)          | Адресація, маршрутизація, передача пакетів             | IP, ICMP, ARP, RIP, OSPF, BGP                                               | **Internet**                  |
| **2. Data Link** (Канальний)        | MAC-адреси, кадри, доступ до середовища                | Ethernet, PPP, Switch                                                       | **Network Access**            |
| **1. Physical** (Фізичний)          | Сигнали, кабелі, радіохвилі, бітові потоки             | Кабель UTP, оптоволокно, Wi-Fi                                              | **Network Access**            |

---

### 🔹 Ключові моменти для інтерв’ю:

- **OSI** = модель-референс (7 рівнів, теоретична, деталізована).
    
- **TCP/IP** = практична модель Інтернету (4 рівні).
    
- Рівні **Application (TCP/IP)** включають одразу **Application + Presentation + Session (OSI)**.
    
- У нижній частині **Network Access (TCP/IP)** = **Data Link + Physical (OSI)**.
