3   Теги: [[Каталоги в системе Linux]]

| Каталог     | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| /           | Корневой каталог, откуда все начинается                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| /bin        | Содержит двоичные (binaries) файлы (программы), необходимые для за-<br>грузки и функционирования системы                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| /boot       | Содержит ядро Linux, образ начального RAM-диска (с драйверами, необхо-<br>димыми на этапе загрузки) и сам загрузчик. Интересные файлы:<br>•/boot/grub/grub.conf или menu.lst, используются для настройки загруз-<br>чика<br>•/boot/vmlinuz (или с похожим именем), ядро Linux                                                                                                                                                                                                                                                                                                      |
| /dev        | Специальный каталог, содержащий узлы устройств. «Все сущее есть<br>файл» применяется также к устройствам. Здесь ядро хранит список всех<br>известных ему устройств                                                                                                                                                                                                                                                                                                                                                                                                                 |
| /etc        | Каталог /etc содержит все системные конфигурационные файлы. Здесь же<br>хранится коллекция сценариев командной оболочки, запускающих систем-<br>ные службы во время загрузки. Практически все файлы в этом каталоге<br>содержат обычный читаемый текст.<br>В /etc все файлы интересны, но, на мой взгляд, особенный интерес пред-<br>ставляют:<br>•/etc/crontab, файл, определяющий время запуска автоматизированных<br>заданий;<br>•/etc/fstab, таблица устройств хранения и соответствующих им точек<br>монтирования;<br>•/etc/passwd, список всех учетных записей пользователей |
| /home       | В обычных конфигурациях каждому пользователю выделяется свой до-<br>машний каталог в каталоге /home. Простые пользователи могут записы-<br>вать что-нибудь только в файлы, находящиеся в их домашних каталогах.<br>Это ограничение защищает систему от необдуманных действий пользова-<br>телей                                                                                                                                                                                                                                                                                    |
| /lib        | Содержит файлы разделяемых библиотек, используемых основными си-<br>стемными программами. Они напоминают библиотеки DLL в Windows                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| /lost+found | Каждый раздел или устройство, отформатированные с использованием<br>файловой системы Linux, такой как ext3, будут иметь этот каталог. Он<br>используется на случай частичного восстановления повреждений в фай-<br>ловой системе. Если с системой ничего страшного не происходило, этот<br>каталог будет оставаться пустым                                                                                                                                                                                                                                                         |
| /media      | В обычных конфигурациях каждому пользователю выделяется свой до-<br>машний каталог в каталоге /home. Простые пользователи могут записы-<br>вать что-нибудь только в файлы, находящиеся в их домашних каталогах.<br>Это ограничение защищает систему от необдуманных действий пользова-<br>телей                                                                                                                                                                                                                                                                                    |
| /mnt        | В старых системах Linux каталог /mnt содержал точки монтирования съем-<br>ных носителей, монтируемых вручну                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| /opt        | Каталог /opt используется для установки «необязательного» (optional)<br>программного обеспечения, в основном для установки коммерческого про-<br>граммного обеспечения                                                                                                                                                                                                                                                                                                                                                                                                             |
| /proc       | Специальный каталог. Не является фактической файловой системой,<br>в том смысле, что файлы в этом каталоге не хранятся на жестком диске.<br>Это виртуальная файловая система, поддерживаемая ядром Linux. Файлы<br>в ней являются «глазками», через которые можно заглянуть в ядро. Эти<br>файлы доступны для чтения и помогают «увидеть» компьютер глазами<br>ядра                                                                                                                                                                                                                |
| /root       | Домашний каталог пользователя root                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| /sbin       | Каталог содержит системные двоичные файлы (system binaries). Эти<br>программы выполняют жизненно важные задачи и обычно запускаются<br>только суперпользователем                                                                                                                                                                                                                                                                                                                                                                                                                   |
| /tmp        | Каталог /tmp играет роль временного хранилища для временных файлов,<br>создаваемых разными программами. В некоторых конфигурациях этот<br>каталог принудительно очищается при каждой перезагрузке системы                                                                                                                                                                                                                                                                                                                                                                          |
| /usr        | Дерево каталогов /usr является, пожалуй, самым объемным в системе<br>Linux. В нем хранятся все программы и файлы поддержки, используемые<br>обычными пользователями                                                                                                                                                                                                                                                                                                                                                                                                                |
| /var        | За исключением /tmp и /home, все упоминавшиеся выше каталоги остают-<br>ся относительно статичными; то есть их содержимое почти не меняется.<br>Дерево каталогов /var — как раз то место, где хранятся часто изменяемые<br>данные: различные базы данных, буферные файлы, почта пользователей<br>и пр.                                                                                                                                                                                                                                                                             |
