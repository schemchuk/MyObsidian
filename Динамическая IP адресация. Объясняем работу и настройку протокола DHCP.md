основи налаштування DHCP сервера
після стандартних налаштуваннь мережі (роутера і сервера) переходжу до налаштуваннь сервера DHCP
для цього на сервері заходжу в нього і переходжу на вкладку 
**services** -> DHCP -> on
Default Gateway   192.168.101.1     **вказую інтерфейс fa 0/0 маршрутизатора R0**
далі задаю початкове значення ІР адрес нашого DHCP сервера:
Start IP Address 192.168.101.100
вказую кількість адрес, які будуть роздаватися автоматично:
Maximum Number of data **10** (довільно) 
для збереження налаштуваннь нажимаю кнопку **save**
потім потрібно налаштувати компютери, щоб вони могли отримувати IP адреси від DHCP сервера(нажати кнопку DHCP в менюшці де раніше задавав настройки вручну )
