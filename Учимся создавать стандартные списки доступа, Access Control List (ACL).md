після  всіх налаштувань(пристроїв та протоколу RIP) мають пінгуватися  всі компиЮ а також всі сервери мають бути доступними для всіх компів.

тепер потрібно зробити:
- щоб компютери з різних [[VLAN]] не були доступні між собою
- щоб сервери були доступні лише окремим компам.
**R0**
створити списки доступів - acxesі lists
VLAN1
ip access-list standard STVlan101  (STVlan101 задаю сам осмисленну назву)
далі вказую його налаштування 
deny 192.168.102.0 0.0.0.255   (заборонив трафік з мережею 192.168.102.0 її зворотня маска 0.0.0.255)
аналогічно:
deny 192.168.103.0 0.0.0.255
**далі я маю вказати небажані для спілкування сервери**
deny host 192.168.201.11
deny host 192.168.201.12
більше заборон **нема**
permit any
do wr
ex

## переходжу до привязки створених **access-list** до заданих інтерфейсів( чи в моєму випадку підінтерфейсів)
**R0**
int fa 0/0.101
ip access-group STVlan101 out       (фільтрфція виходящого(**out**) трафіку)
do wr
ex
int fa 0/0.102
ip access-group STVlan102 out       (фільтрфція виходящого(**out**) трафіку)
do wr
ex
int fa 0/0.103
ip access-group STVlan103 out       (фільтрфція виходящого(**out**) трафіку)
do wr
