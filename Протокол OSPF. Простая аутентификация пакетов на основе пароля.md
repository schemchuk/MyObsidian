для підвищення захищеності інформації протоколи OSPF можна зашифрувати паролями
будую мережу і роблю первинне налаштування приладів  а потім налаштування протоколу OSPF
далі **R1** звертаюся до інтерфейсу через який буде проходити обмін службовою інформацією
int fa 0/1
ip ospf authentication-key 123
do wr
повертаюся в реж гл конф і зашифровую ключ
exit
service password-encryption
do wr
router ospf 1
area 0 authentication
do wr
аналогічні налаштування (потрібно виконувати в обох інтерфейсах на **R2** тому що в передачі служб інформації будуть задіяні оба) виконую на **R2** i **R3** **паролі повинні бути однакові на одному інтерфуйсі** на різних інші

далі **R2** звертаюся до інтерфейсу через який буде проходити обмін службовою інформацією
int fa 0/1
ip ospf authentication-key 123
int fa 0/0
ip ospf authentication-key 456
do wr
повертаюся в реж гл конф і зашифровую ключi
exit
service password-encryption
do wr
router ospf 2
area 0 authentication
do wr
