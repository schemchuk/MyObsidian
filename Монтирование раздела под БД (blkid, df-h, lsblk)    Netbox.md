3   Теги: [[Монтирование раздела под БД (blkid, df-h, lsblk)    Netbox]]
додаю в вірт боксі ще один жорсткий диск до віртуальної машини (називається машина Розділи)
заходжу в вірт машину під рутом

Подивитися що примонтовано
df -h
подивитися які диски є в файловій системіва
lsblk
підтягую  утиліту для роботи з дисками parted (альтернатива fdisk)
apt install -y parted
подивитися всі доступні устройства:
parted -l

працюю з новододаним диском sdb
parted  /dev/sdb
## створюю таблицю  розділів
mklabel [[GPT]]
подивитися інформацію про устройство з яким працюєм
print
##  створити новий розділ
mkpart database ext4 1MB 7,6GB   (задіюю весь простір диска)
print
виихід q
q
переконатися що розділ  sdb примонтовано
lsblk
можна ще подивитися в /dev
cd /dev
## відформатувати розділ  sdb
	mkfs -t ext4 /dev/sdb1
## примонтувати устройсво
cd /
mkdir -p pstgsql
mount /dev/sdb1 /pstgsql

перевірити

cd /pstgsql
ls
### роблю щоб розділ sdb примонтовувався автоматично при старті системи
vim /etc/fstab
добавляю строку 
/dev/sdb1 /pstgsql ext4 defaults 0 0
 якщо через UUID
 (якщо хочеш тільки чисте значення UUID без зайвого):
 
  blkid -s UUID -o value /dev/sdb1 root

підставляю значення і записую в файл fstab
UUID="1742d99e-cd71-9e16-52c1da048a9f"  /pstgsql ext4 defaults 0 2
